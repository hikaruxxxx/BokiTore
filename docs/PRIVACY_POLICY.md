# プライバシーポリシー — 簿記トレ

最終更新日: 2025年3月

## 1. 収集するデータ

### 1.1 Firebase Analytics（匿名利用状況データ）
本アプリでは、サービス改善のためにGoogle Firebase Analyticsを使用し、以下の匿名データを収集しています：

- アプリの利用状況（起動回数、セッション時間、画面遷移）
- 問題の解答状況（正誤率、カテゴリ別成績）※個人を特定できない統計データ
- デバイス情報（OS バージョン、デバイスモデル）
- 地域情報（国・地域レベル）

これらのデータは個人を特定できない形で収集され、Googleのサーバーで処理されます。

### 1.2 ローカル学習データ
以下のデータはお使いのデバイス内にのみ保存され、外部に送信されることはありません：
- 学習履歴（解答した問題、正誤、回答時間）
- 連続学習日数
- 設定情報（通知時刻、ダークモード等）

### 1.3 広告関連（Google AdMob）
本アプリでは広告配信のためにGoogle AdMobを使用しています：
- AdMobは広告表示のためにデバイス識別子を使用する場合があります
- App Tracking Transparency（ATT）の設定に基づき、パーソナライズ広告の有無が決まります
- 詳細はGoogleのプライバシーポリシーをご確認ください

## 2. データの使用目的

収集したデータは以下の目的にのみ使用します：
- アプリの品質・ユーザー体験の改善
- 学習コンテンツの改善（出題傾向の分析）
- 広告の配信
- アプリの安定性向上（クラッシュ分析）

## 3. 第三者への提供

個人を特定できるデータを第三者に販売・提供することはありません。
Firebase AnalyticsおよびAdMobのデータ処理については、Googleのプライバシーポリシーに準じます。

## 4. データの保持期間

- Firebase Analyticsのデータ: Googleの標準保持期間に準拠
- ローカル学習データ: アプリの削除まで端末内に保持

## 5. お子様のプライバシー

本アプリは13歳未満のお子様を対象としたサービスではありません。

## 6. お問い合わせ

プライバシーに関するお問い合わせは、App Storeのアプリページに記載のサポートURLからご連絡ください。
