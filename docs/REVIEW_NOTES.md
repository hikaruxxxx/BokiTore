# Apple審査チーム向けメモ — 簿記トレ

## アプリ概要

簿記検定3級の仕訳問題を演習できる学習アプリです。
4択形式の問題に回答し、解説を読んで理解を深める形式です。

## 主な機能

1. **問題演習**: カテゴリ別・難易度別の仕訳問題（50問以上）
2. **解説表示**: 各問題に日本語の解説付き
3. **学習統計**: 正答率、学習日数、カテゴリ別成績
4. **ストリーク**: 連続学習日数の記録
5. **デイリーチャレンジ**: 毎日の学習目標
6. **通知リマインド**: ユーザー設定の時刻に学習リマインド

## 広告SDKの使用

- **Google AdMob** を使用しています
  - バナー広告: 各画面下部に表示
  - インタースティシャル広告: 10問回答ごとに表示
  - テスト広告IDは開発時のみ使用し、本番ではAdMobコンソールで発行したIDを使用
- **Firebase Analytics** を使用しています
  - 匿名の利用状況データのみ収集
  - 個人を特定できる情報は収集しません

## アプリ内課金

- **プレミアムプラン（月額自動更新サブスクリプション）**
  - 広告の完全非表示
  - 全カテゴリの詳細分析機能
- StoreKit 2 を使用
- 購入の復元機能あり（設定画面から実行可能）

## プライバシー

- ユーザーの学習データはデバイスローカルにのみ保存
- 外部サーバーへの個人情報送信なし
- Firebase Analytics は匿名統計データのみ収集
- ATT（App Tracking Transparency）対応済み

## テスト用アカウント

- アカウント登録は不要です
- アプリを起動するとすぐに使用できます

## 特記事項

- 「日商簿記」は日本商工会議所の登録商標のため、アプリ説明文では「簿記検定対応」と表記しています
- 問題内容はオリジナルで作成しており、著作権上の問題はありません
- すべてのコンテンツは日本語です
